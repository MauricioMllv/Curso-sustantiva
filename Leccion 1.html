1-Se entiende por desarrollo web el proceso de crear y mantener sitios o aplicaciones web, implica variadas tareas entre ellas diseño web, front end, back end.

2- Diferencias entre Front-end, back-end y fullstack

Front end
Se encarga del aspecto visual de los usuarios
Asegura que la interfaz sea intuitiva y atractiva
Optimiza y se encarga de compatibilidad tanto de navegadores como dispositivos

Back end
Se encarga de la parte no visible de un sitio web
Se encarga de la seguridad de la manipulación de datos
Gestiona el rendimiento en cuanto a escalado para un mayor trafico

Fullstack
Maneja ambas partes tanto front end como back end por lo cual contiene las responsabilidades de ambos

4- Un navegador interpreta un documento HTML mediante el análisis sintáctico para crear el Modelo de Objeto del Documento (DOM). Luego construye el Árbol de Renderizado, aplica estilos para mostrar la página web.

5- Que es la W3C

World Wide Web Consortium, establece estándares y pautas para la World Wide Web, asegurando el desarrollo sostenible de la web.

6-HTML -> HTML5
